# Daily Planner

2열 레이아웃 데일리 플래너 UI 프로토타입 (HTML + CSS)

## 실행 방법

1. VS Code에서 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 확장 설치
2. `index.html` 우클릭 → **Open with Live Server**
3. 브라우저에서 UI 확인

또는 터미널에서:

```bash
npx --yes serve .
```

## 현재 상태

- 왼쪽: 할 일 목록 (더미 데이터)
- 오른쪽: 04:00 ~ 24:00 시간표 + 샘플 블록
- JavaScript 기능 없음 (UI 목업만)
